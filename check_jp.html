PADI マーメイドコース案内

自然に抱かれるコバルトブルーの旅
PADI ディスカバー マーメイド PADI ベーシック マーメイド PADI マーメイド PADI アドバンスド マーメイド

太陽・ガラスのような海・マーメイドテール――沖縄がくれる三つの夢。

沖縄は透明度の高い“ガラスの海”、眩しい日差し、多彩なサンゴ礁が揃う、アジア屈指のマリンフィールド。
RiShima Dive では、マーメイドに憧れるすべての方のために PADI マーメイド系統を段階的にご用意。
初めての体験から本格的なスキルアップ・資格取得まで、ゼロから夢の尾びれをまとい、本物の水中マーメイドへと進化していけます。
旅の特別な記念にも、将来のパフォーマンス / 指導の第一歩にも、ここがあなたの“はじまりの海”。

“見る海” から “海の一部になる” へ――沖縄でその一歩を。


1. PADI ディスカバー マーメイド｜PADI Discover Mermaid

沖縄に来たらまずはテールを履こう！夢の第一歩、未経験でも安心。

経験ゼロでOK。国際認定インストラクターが、専用テールの装着から基本のフロート、ドルフィンキックを丁寧にサポート。
浅いプールで安全・快適に“人間からマーメイドへ” の変化を体感します。
旅の思い出づくりにも、童話世界のプチ体験にも最適。
→ 詳細を見る


2. PADI ベーシック マーメイド｜PADI Basic Mermaid

体験から一歩前へ。沖縄の蒼で “基礎力” を固める。

呼吸コントロール、安定したフロート、方向転換、推進と軽い潜降など、体系立てて学ぶ入門資格コース。
自信を持ってテールを扱う感覚を養い、次のステップ “PADI Mermaid” への土台を築きます。
→ 詳細を見る


3. PADI マーメイド｜PADI Mermaid

沖縄の“ガラスの海” があなたのステージへ。

長距離キック、ホバリング（静止）、エレガントなターンなど高度な動作を強化し、環境適応やセルフマネジメントも習得。
資格取得後は、より表現力豊かな撮影やパフォーマンスに挑戦できるレベルへ。
→ 詳細を見る


4. PADI アドバンスド マーメイド｜PADI Advanced Mermaid

プロフェッショナル表現・指導・出演へのゲートを開く。

高度な水中姿勢・持久力・チームワーク・水中振付（コレオ）・撮影対応を包括的に磨く 2 日プログラム。
将来インストラクターや水中パフォーマーを目指す方の重要ステップ。
→ 詳細を見る
